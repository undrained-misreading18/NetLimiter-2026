## 🛡️ Using the traffic filters

Once you launch the program, you see a list of every active process that uses your network. Each entry shows the current upload and download speed for that specific process. 

To limit a program, find it in the list. Right-click the program name. You will see options to set a limit. Input the speed in kilobits or megabits per second. Save these settings. The program will now follow your speed limit.

To block a program, use the firewall features. Look for the small icons next to the application name. A lock icon represents internet access. Click this icon to stop the application from connecting to the web. Click it again to restore access.

## ❓ Frequently asked questions

Do I need admin rights to install this? Yes, you need administrator access on your Windows account to install the software. This is because the tool interacts with network drivers.

Can I block all internet traffic at once? Yes, you can use the Global Traffic rule. This stops all applications from using the network. Use this feature with care.

Will this software slow down my computer? No, the tool runs in the background with low resource usage. It does not affect your CPU performance during daily tasks.

Where does the software save my settings? It saves your rules automatically. If you close the program and open it later, your limits will still be active.

## 📈 Troubleshooting tips

If the program lists no data, try to restart the NetLimiter service. Open your Task Manager. Look for Services. Find the NetLimiter service in the list of names. Right-click it and select Restart. 

If you see an error during installation, check your firewall. A restrictive firewall might block the installer. Disable your existing antivirus software for a few minutes while you complete the installation. Remember to turn your antivirus back on once you finish the setup.

If you encounter issues with speed, check your router settings. Sometimes hardware limits conflict with software limits. Test your connection without the software to see if the speed returns to normal.
